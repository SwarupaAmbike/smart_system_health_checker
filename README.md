# smart_system_health_checker
ML project

##  Description
This is a beginner-level machine learning project that predicts whether a computer system is healthy or at risk using basic system parameters.

##  Objective
To identify potential system issues early using simple machine learning techniques.

##  Input Parameters
- CPU usage
- Memory usage
- Temperature
- Crash count

##  Output
- System is healthy
- System is at risk

##  Model Used
- Decision Tree Classifier

##  Technologies Used
- Python
- Pandas
- Scikit-learn

##  How it Works
The model is trained on sample system data and predicts the risk level for new system inputs.

##  Future Improvements
- Use real-time system data
- Add more system parameters
- Build a dashboard using Streamlit
- Explore TensorFlow for advanced models

##  Status
This is a beginner-level project created to understand machine learning concepts and real-world applications.
